# Comprobac

Comprobac is a Flutter application designed to validate the veracity of bank transfer references. It includes features to help users prevent fraud in bank transactions.

## Features

- **Login Page**: User authentication.
- **Onboarding Page**: Information on preventing fraud in bank transactions.
- **Home Page**: Menu with an option to validate transactions.
- **Validate Transaction Page**: Form with fields:
  - Nº Comprobante
  - Nº Cuenta Acreditarse
  - Seleccionar Mes

## Dependencies

- `riverpod`
- `flutter_localizations`
- `spider`
- `bancadigital_bm_red_designsystem` (hosted at https://gsbd.jfrog.io/artifactory/api/pub/digital-lab-pub-local, version ^2.2.0-alpha.5)

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to fetch dependencies.
3. Launch the application using `flutter run`.
